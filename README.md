# DeadUnlock Hybrid

External cheat for Deadlock (Source 2) - C++ with Python dumper.

## 🏗️ Architecture
- **Python Dumper**: Finds offsets via signature patterns
- **C++ Cheat**: External overlay with ImGui menu
- **Anti-Flag System**: Humanization to avoid detection

## 🔧 Setup

### Prerequisites
- Windows 10/11
- Visual Studio 2022 with C++
- Python 3.8+ (`pip install psutil pymem`)
